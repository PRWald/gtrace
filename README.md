# gtrace

Python modules and scripts for working with Garmin trace files. With it you
can plot Garmin traces over the freely available maps from OpenStreetMap.

## Python modules
- garmin.py
- map_tile_mgr.py
- waypoint_mgr.py
- extract_exif_gps.py

## Scripts
- csv2pathmatrix.py
- csv2paths.py
- plot-gps-traces.py
- trace2csv.py
- trace2waypoints.py
- trace_analyzer.py
- trace_explorer.py
- traces-stats.py

## Resource File

User needs to create a resource file with the following definitions:
1. map_api_key
2. tile_cache
3. waypoints_file
4. path_csv_file
